# CODSOFT Task 1 - Number Guessing Game

This task is a Number Guessing Game developed in Java.

## Features
- User selects difficulty (Easy, Medium, Hard)
- Random number is generated
- User keeps guessing until the correct number
- Program gives hints (High / Low)
- Simple and interactive Java application
- limited attempts
- display the user score

## Concepts Used
- Java basics (variables, loops, conditions)
- Random number generation
- Scanner for input
